## Installation
[available in Hex](https://hex.pm/packages/parasut), the package can be installed
by adding `parasut` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:parasut, "~> 0.1.0"}
  ]
end
```
